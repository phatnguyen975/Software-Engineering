## Course Description

This course is to introduce to students basic concepts, principles, and practice in software engineering. Students will learn different types of software process models, such as Waterfall, RUP, Agile, etc. Main phases and roles in software process will be discussed. Student will have chances to practice writing software specifications, architecture & design models, test plans, etc. Students will also use tools and technologies to build practical software project in team.
