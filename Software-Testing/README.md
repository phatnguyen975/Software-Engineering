## Course Description

This course covers the foundations of software testing and AI-augmented testing: the testing process, test case design techniques (black-box, white-box, domain, state, decision table, use case), automation, performance, API/CI-CD, GUI/usability, security, DB testing, and mutation testing. The Project and most Homeworks are run against a single shared System Under Test - the EShop demo e-commerce application maintained by the Department. Students are required to use AI tools (of their own choice) for any task where AI is appropriate, and
to audit each AI-generated artefact rigorously using the `[AI-01]`..`[AI-06]` template pack.
